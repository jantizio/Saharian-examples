presentation examples

Here you can find different examples in which assistive technologies gets informed that they should completely ignore the semantics of an element.

Plain HTML

O well... HTMl does not provide any native mechanism to remove semantics from its elements.

Correct ARIA presentation

In this example the "presentation" role from the WAI-ARIA specification is used to remove any semantics from a button.

Abusing of the presentation role

In this example the "presentation" role from the WAI-ARIA specification is used to remove any semantics from a clickable link. The result is that assistive technology users are not informed that they can interact with this link.
